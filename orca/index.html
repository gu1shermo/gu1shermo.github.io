Orca Livecoding

A creative programming companion for real-time performances on Guillaume Cournet's site

Introduction

Orca is a livecoding environment designed to enable creative performances in real-time. With a focus on simplicity and flexibility, Orca allows you to experiment with code and create engaging experiences.

Features

  • Real-time coding and performances
  • Flexible and expressive coding environment
  • Supports various creative programming techniques
  • Customizable for different types of performances

Getting Started

To start using Orca, follow these simple steps:

  1. Download Orca from the official site.
  2. Set up your environment according to the provided instructions.
  3. Explore the documentation and examples to learn how to use Orca effectively.
